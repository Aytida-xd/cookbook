# Welcome to the Smallest AI Cookbook

This cookbook provides a structured learning path for using the Smallest AI APIs and SDKs, focusing on hands-on tutorials and practical examples.

## Getting Started

1. **Get an API Key**: Sign up at [smallest.ai/console](https://smallest.ai/console)
2. **Set Environment Variable**: `export SMALLEST_API_KEY="your-key"`
3. **Choose a Cookbook**: Pick one that matches your use case
